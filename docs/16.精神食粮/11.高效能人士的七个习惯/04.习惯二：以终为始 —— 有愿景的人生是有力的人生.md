## 习惯二：以终为始 —— 有愿景的人生是有力的人生

::: tip 现在，请你清空头脑，最好能闭上眼睛，在脑海中想象以下的场景：

假设你正在前往殡仪馆的路上，去参加一位至亲的葬礼。你缓缓走进告别室，淡淡的花香伴随着肃穆的音乐。一路上你见到很多亲友熟悉的面孔，你能体会到他们痛失至亲的悲哀，也能回想起往日相聚的欢乐。

到达前厅，当你看向棺木时，你发现面对的竟然是你自己！是的，没有错，这就是你自己的葬礼，也许这发生在很久之后，也许，并不那么的遥远。

今天，你的亲朋好友们从各地齐聚一堂，追述你的生平，表达对你的爱和感激。一共有三位致辞的嘉宾，第一位是你的家庭成员，ta 可能是你的配偶、子女、兄弟姐妹。第二位是你的好朋友，第三位是你工作和事业中的伙伴。现在请仔细想想，你希望 ta 们怎样描述你的一生呢？

在 ta 们眼中你是个怎样的丈夫、妻子、父亲、母亲？又是个怎样的朋友和同事？你希望 ta 们说你是个怎样的人呢?能提到你的哪些成就？因为你的存在给 ta 们的生活带来了什么不同？请想想你的回答，这些将帮助我们更好地理解习惯二。

:::

### 『 以终为始 』的含义

习惯二以终为始，来自英文 `Begin with the end in mind`，直接翻译过来就是『 开始的时候脑海里就有终点的画面 』，大体上可以理解为『 采取行动前要想清楚目标 』。不过，在七个习惯中，『 以终为始 』的重点并不在这个层级，而是指：**开始每一天的时候，在脑海里想一想在生命终点时你所希望的画面，并以此来指导生活中的所有事情。**

那么为什么非要谈到『 生命终点 』呢？毕竟这在全世界很多文化中都是最为禁忌的话题。原因就在于：当人们赤裸裸地面对生命终点时，才有可能彻底抛开来自外部的所有期待值，毫无顾忌地表达自己内心深处最渴望的东西。这实际上是一种『 向死而生 』的人生哲学，日常生活中可能很少会遇到。那么有没有人真的在这样做呢？有的！让我们一起听一段2005年美国斯坦福大学毕业典礼上的著名演讲。

演讲者正是苹果公司创始人史蒂夫.乔布斯。『 记住我即将死去 』是我一生中遇到的最重要的箴言，它帮我做出了生命中重要的选择。因为几乎所有的事情，包括所有外部的期待，所有的荣耀，所有的尴尬或失败，这些在死亡面前都会消失，最重要的。你有时候会陷入患得患失的迷局，『 记住你即将死去 』是我知道的保持清醒的最好方法。你本来就无可失去，没有理由不去追随本心。 』这段话可以说是对习惯2『 以终为始 』的生动诠释。

人生在世，没有人能脱离环境的影响，各种来自他人的期待值和形形色色的社会思潮混合在一起，包围着每一个人。柯维博士把这种现象称为『 社会之镜 』：我们好像身处装满镜子的屋子，看到的其实是镜子里的反射，让人头晕目眩。而以终为始，就像乔布斯所说，帮我们找回真正的自己，留下真正重要的。要回答好这个问题，不能向外看，而是要向内看。

刚才的葬礼实验，就是一个向内觉察的体验，好比把那些明晃晃的社会之镜都拆除了，一个人内心最深处的渴望慢慢浮现出来，这是ta自己对『 什么是成功生活 』的回答。人与人是不同的，对有的人来说是商业中取得丰功伟绩，对另一些人可能是经历几段刻骨铭心的感情，对有的人可能是走遍地球的每个角落，还有的人可能是把子女们养育成人。不管内容是什么，都是一种生命的召唤，它召唤人们无怨无悔地付出，同时也让人们内心产生『 充实 』、『 有意义 』的感受。这个付出的过程可以激发人们的智慧和力量，赋予人们做出选择的勇气，有助于保持振奋的精神状态，从生活中获得更高的个人满足感。

比如乔布斯，我们不知道他内心认为的『 真正重要的是 』是什么，不过他曾经在多个场合公开表达对生命的看法：**活着就是为了改变世界，难道还有其他原因吗？** 这个声音召唤者他，给他希望和智慧，做出重大的选择，也有力量面对重大的人生挫折。乔布斯的答案当然不是标准答案，你我的答案也可能与此不同。不过我们可以看到，不管这个答案是什么，有一个属于自己的清晰的答案，更可能度过一个精彩的人生。

在习惯一主动积极中，我们介绍了奥地利心理学家弗兰克尔的故事。他根据在死亡集中营的经历，后来发展出了一种精神治疗方法，叫做 `logotherapy`，可以翻译成『 意义疗法 』。其主要思想，就是认为现代人的很多精神疾病就是来自于内心的空虚、无意义的感觉，从而引发倦怠、抑郁等各种疾病。而意义疗法，就是帮助人们自己找到心中的意义感，然后那些症状就会随之而自动消失。这个正是习惯2以终为始的重要作用。

在咱们得社会中，我们经常看到一些现象，比如咱们国家的很多年轻人在学生时代并没有形成这种内心的成熟度，是为了父母而读书，为了高考而读书，就像一件文化衫上写下的调侃文字：我爱学习，学习使我妈快乐。他们一旦进入大学，就立刻失去了意义感，内心变得极度空虚无聊。很多人不但荒废了时光，还滋生出种种精神方面的问题。

好消息是，越来越多的年轻人有了清晰地自我意识和以终为始的思维方式。我作为讲师，曾经给一些初中和高中的青少年讲解七个习惯，就发现跟我们那代人比，他们的进步非常明显，在谈到自己的生活愿景时，几乎没有一个孩子含糊其辞，浑浑噩噩，而是都能清楚、明确的表达自我，比如『 我要做一名研究热带雨林昆虫的生物学家 』，『 我要建立一个有两个孩子的幸福家庭 』，『 我要成为中国第一批前往火星的宇航员 』。我由衷的为这些孩子高兴，相信在使命的召唤下，他们的人生会更加精彩。

### 『 以终为始 』实践方法：个人使命宣言

上面这些个人心声，也是实践以终为始的最实际的方法，柯维博士把它成为『 个人使命宣言 』。

**『 使命 』**这个词在现代汉语里应该说并不是一个很常用的词汇，使命的本意就是指**某一个行程所肩负着的任务**。如果我们把人生看成一个旅程，那么个人使命就是在人生旅途中所肩负的主要任务。这个任务是什么呢？同一，答案不在外面，而在每个人心中。就像弗兰克尔所说：**我们发现而不是发明了自己的人生使命。**他的意思就是每个人的使命本来就是存在的。而不是凭空捏造出来的。

使命感能够让人们超越小我，把自己和某个比自己大得多、重要得多的任务联系起来，把自己看成是这个任务的一个载体，这样的意义在于，有使命的人内心更加有安全感，更能够被激发出力量和智慧。同时，面对困难时，使命感就好像是一个人内置的GPS导航系统，可以帮助人们做出自己的选择。


::: tip 从生物进化论的角度看

其实每一种生物都带有一种天然的使命，那就是生存和繁衍。比如鲑鱼，也就是大马哈鱼，为了完成繁衍的使命，每年从太平洋海域开始几千公里的旅行，回到出生地产卵，在河流中要逆流而上，中途要跳过小瀑布在这个壮观的旅途中他们表现出了极大地勇气和力量。再比如地球上迁徙距离最长的动物——北极燕鸥，为了在北半球的夏季回到北极产卵，它们绕地球半圈儿，飞行里程长达两万公里。这个人物赋予北极燕鸥一种神奇的导航系统，在漫长的长途飞行中永远不会迷失方向。所以说使命激发力量，使命指引方向。

:::

再来看人类，除了完成所有生物共同的使命以外，人因为具有动物所不具有的独特天赋，比如想象力和独立意志，就可以最求更高层次的使命。古今中外这样的人物有很多。比如说袁隆平先生，终生以研究杂交水稻为使命。在他年轻的时候曾经在路边看到有两具饿死的尸体，从此他就把自己交付给了利用技术创造更高的粮食产量这个任务。在2019年9月29日，他获得了国家主席颁发的『 共和国勋章 』，接受采访时他说『 每晚睡觉时都在想我的超级水稻长得怎么样了。领了勋章回去第一天我就要去下田。 』再比如中国唐代的高僧玄奘，从二十多岁就把自己交付给获得佛学原始经典的任务，他西行佛学的发源地天竺，往返几万里，历经十七年。这个使命给他力量和勇气，克服了很多九死一生的磨难，最终实现心愿。再比如当代的美国企业家，特斯拉公司的联合创始人埃隆马克思从三十岁起就有了实现人类移民火星的梦想。他为此而创立了spaceX太空技术公司，多次经理火箭发射失败，公司拼林倒闭的边缘。不过他的梦想依然在激发着众多行业内的高手，预计最早于2020年他们会想火星发射飞船。

当然，跟这些已经名垂青史的人比，我们绝大多数人都是普通人，过着平凡的日常生活，不过，我们也要看到，他们的成就最初就是来自内心深处朴素的、真切的渴望，是完成个人使命过程最大化激发了智慧和勇气。最终实现了非凡的成就。

从这个角度讲，每个普通人都可以，也应该拥有个人使命，因为他可以让我们的内心更充实，度过让自己更满意的人生。柯维博士说：个人使命宣言就像是一部个人宪法，他指导我们做出生活中的重大决定。德国哲学家尼采也说过：知道『 为什么活 』的人，永远能回答『 该怎么办 』的问题。有使命感的人，不容易在社会中迷失自己。

个人使命宣言的撰写并不是一蹴而就的，需要深刻的自我觉察，想象力的调动和深思熟虑的表达方式，过程中可能有多次修改。一旦形成，就是一个稳定的版本。虽然随着环境和个人生活状态的变化，有微调的可能。但时间间隔通常都是以几年为单位，就好比一个国家的宪法一旦形成，绝对不会经常修订。
