## 一、一个简单工程

> 以 SpringBoot 工程为例：

### 1. 安装与配置

##### 1.1 依赖

```xml
<dependency>
	<groupId>com.baomidou</groupId>
	<artifactId>mybatis-plus-boot-starter</artifactId>
	<version>3.4.2</version>
</dependency>
```

##### 1.2 配置

1. 启动类：

```java
@MapperScan("com.example.myplusdemo.mapper")
```

2. `application.yml`：

```yml
spring:
  datasource:
    driver-class-name: com.mysql.cj.jdbc.Driver
    url: jdbc:mysql://localhost:3306/hello?serverTimezone=UTC
    username: root
    password: 123456

# Logger Config : 打印 SQL 语句
logging:
  level:
    com.example.myplusdemo: debug

#mybatis:
mybatis-plus:
  mapper-locations: classpath:mapper/*Mapper.xml
  type-aliases-package: com.example.myplusdemo.entity
```

### 2. CRUD

##### 2.1 `Entity`实体类

```java {2,4}
@Data
@TableName("user")
public class User {
    @TableId(value = "id", type = IdType.AUTO)
    private int id;
    private String name;
    private int idCer;
}
```

##### 2.2 `Service`接口

```java {1}
public interface IUserService extends IService<User> {
}
```

##### 2.3 `Service`实现类

```java {1}
@Service
public class UserServiceImpl extends ServiceImpl<UserMapper, User> implements IUserService {
}
```

##### 2.4 `Mapper`类

```java {1}
public interface UserMapper extends BaseMapper<User> {
}
```

##### 2.5 测试类

```java
@SpringBootTest
class MyPlusDemoApplicationTests {

    @Resource
    private UserMapper userMapper;

    @Autowired
    private IUserService userService;

    @Test
    void contextLoads() {
        System.out.println(("----- selectAll method test ------"));
        List<User> userList = userMapper.selectList(null);
        userList.forEach(System.out::println);
        System.out.println("================ service: ================");
        List<User> users = userService.list();
        users.forEach(System.out::println);
    }

}
```
